# NARRATIVE — ATOM-111 (проектный маршрутизатор)

> Живая лента смысла. Дописывается только в конец (INFO-015/016).

**10.07, старт по «го» (GATE-029).** Твой вопрос «как един бот, если
проектов несколько?» попал в архитектурную дыру v1 — и стал деревом. Цель:
один бот остаётся одним голосом для всех проектов машины — события с
ярлыком «[проект]», решения уходят домой, дайджест один со секциями, мысль
без адреса получает один переспрос кнопками. Жёсткое условие — твой живой
соседний мак обновляется молча: без повторной привязки, без второго «я на
связи»; заодно лечится сегодняшний дефект доката. Исполнитель — тот же
toolsmith (шестой прогон под дисциплиной зрелости), конверт ~250k из ~450k
дерева.
